# 兼容性报告
## 太极

### 状态  
   - **Riru - InternalBrowserRedirect** 与 **太极 - Magisk** 不兼容  
  
### 行为  
   - **太极 - Magisk** **总是** 覆盖 Riru - InternalBrowseRedirect 的 钩子  
  
### 解决方案
   - 停用 **太极 - Magisk**
   - 尝试 [EdXposed](https://github.com/ElderDrivers/EdXposed) 代替 **太极**
